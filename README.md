# Spotify Data Analysis
## Overview

This project focuses on analyzing music data from Spotify to identify trends in track popularity, album distribution, and listener behavior.

## Objectives
-Analyze track popularity trends
-Understand album vs single distribution
-Study explicit content proportion
-Identify release year trends
-Explore artist representation
## Key Insights
-68.24% of tracks belong to albums, dominating the catalogue
-Singles account for 26%, while compilations have minimal share
-Only a small proportion of tracks achieve high popularity
-Recent releases (post-2000) show significantly higher engagement
-25% of tracks are explicit, while the majority are non-explicit
## Tools & Technologies
Python (Pandas, NumPy)
Seaborn & Matplotlib
Jupyter Notebook
# Conclusion

The analysis highlights the dominance of album-based releases and increasing popularity of recent tracks, driven by digital streaming platforms like Spotify.
